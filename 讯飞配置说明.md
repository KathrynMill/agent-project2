# 讯飞语音识别配置说明

## 🎯 第一步：获取讯飞密钥（您需要做）

### 1. 访问讯飞开放平台
```
https://www.xfyun.cn/
```

### 2. 注册账号
- 使用手机号注册
- 接收验证码登录

### 3. 创建应用
访问控制台：
```
https://console.xfyun.cn/app/myapp
```

点击 **"创建新应用"**：
- **应用名称**：语音助手（随便填）
- **应用平台**：WebAPI
- **应用领域**：随便选一个

### 4. 获取密钥
创建后会显示三个重要信息：

```
APPID：     xxxxxxxx（8位）
APISecret： xxxxxxxxxxxxxxxxxxxxxxxxxxxx
APIKey：    xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

**请复制这三个值！**
